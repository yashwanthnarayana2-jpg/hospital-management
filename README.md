 # 🏥 Intelligent Healthcare Management Platform

A web-based healthcare management system that simplifies hospital operations by enabling secure patient, doctor, and admin management with AI-powered healthcare assistance.

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Role-Based Access (Admin, Doctor, Patient)
- Doctor & Patient Management
- Appointment Booking & Tracking
- AI Healthcare Chat Assistant (OpenAI API)
- RESTful APIs for frontend integration

## ⚙️ Setup
1. Clone the repository
   ```bash
   git clone https://github.com/your-username/intelligent-healthcare-management-platform.git
   ```
2. Configure MySQL and `application.properties`.
3. Run the project:
   ```bash
   mvn spring-boot:run
   ```
